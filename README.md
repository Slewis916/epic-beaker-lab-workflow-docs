# Epic Beaker Clinical Pathology — Workflow Documentation Portfolio

## Overview
This repository contains clinical workflow documentation created from 6+ years of hands-on Epic Beaker CP experience as a Medical Laboratory Scientist in the Virology department at Yale New Haven Hospital, one of the nation's leading Epic implementations.

The documentation in this repository demonstrates analyst-level thinking including test build specifications, interface troubleshooting runbooks, incident management logs, and clinical workflow analysis. All order codes, facility identifiers, and system configurations are fictional and created for portfolio demonstration purposes only. Content does not represent any specific health system's proprietary build configuration.
